// about the project

Built for defenders

Artemis III is a browser-only OSINT tool — no server, no API keys, no installation. Every piece of data it collects is publicly available. The same data attackers use for reconnaissance.

The goal is to let defenders see their own infrastructure the way an attacker would — before any attack begins. A full scan takes under 60 seconds and requires nothing but a domain name.

Built as an academic project exploring the intersection of open-source intelligence, cybersecurity, and modern web development.
